
# 云开发适配说明

## 概述

本项目已为后续微信云开发做好接口准备，当前使用本地存储模拟数据，可无缝切换到云开发。

## 项目结构

```
test2/
├── config/
│   └── cloud-config.js          # 云开发配置
├── utils/
│   └── data-service.js          # 数据服务层（核心接口）
├── pages/                       # 页面文件
└── app.js                       # 应用入口
```

## 核心文件说明

### 1. config/cloud-config.js
云开发配置文件，包含：
- 环境ID（envId）
- 数据库集合名称配置

### 2. utils/data-service.js
数据服务层，提供统一接口：

#### 存储接口
- `getStorage(key)` - 获取数据
- `setStorage(key, data)` - 保存数据
- `removeStorage(key)` - 删除数据

#### 集合接口
- `getCollection(collectionName)` - 获取集合
- `addToCollection(collectionName, data)` - 添加数据
- `updateCollection(collectionName, id, data)` - 更新数据
- `deleteFromCollection(collectionName, id)` - 删除数据

#### 云函数接口
- `callCloudFunction(functionName, data)` - 调用云函数

## 切换到云开发的步骤

### 步骤1：开通云开发
1. 在微信开发者工具中打开项目
2. 点击顶部"云开发"按钮
3. 创建云开发环境，获取环境ID

### 步骤2：配置云开发
修改 `config/cloud-config.js`：
```javascript
const CloudConfig = {
  envId: '你的环境ID',  // 填入你的环境ID
  collections: {
    users: 'users',
    restaurants: 'restaurants',
    dishes: 'dishes',
    orders: 'orders',
    addresses: 'addresses',
    posts: 'posts',
    merchantOrders: 'merchantOrders'
  }
};
```

### 步骤3：初始化云开发
在 `app.js` 的 `onLaunch` 中添加：
```javascript
onLaunch() {
  this.initCloudDevelopment('你的环境ID');
  // ... 其他代码
}
```

### 步骤4：实现云开发接口
修改 `utils/data-service.js` 中的云开发方法：

#### 云数据库操作
```javascript
getCloudCollection(collectionName) {
  const db = wx.cloud.database();
  return db.collection(collectionName).get().then(res => res.data);
},

addToCloudCollection(collectionName, data) {
  const db = wx.cloud.database();
  return db.collection(collectionName).add({
    data: {
      ...data,
      createTime: db.serverDate()
    }
  });
}
```

#### 云函数调用
```javascript
callCloudFunction(functionName, data = {}) {
  return wx.cloud.callFunction({
    name: functionName,
    data: data
  });
}
```

### 步骤5：创建云数据库集合
在微信云开发控制台创建以下集合：
- users（用户）
- restaurants（餐厅）
- dishes（菜品）
- orders（订单）
- addresses（地址）
- posts（帖子）
- merchantOrders（商户订单）

### 步骤6：创建云函数
根据业务需求创建云函数，如：
- login（登录）
- createOrder（创建订单）
- generateOrderNumber（生成订单号）

## 当前使用示例

### 页面中使用数据服务
```javascript
const app = getApp();
const dataService = app.getDataService();

Page({
  onLoad() {
    const addresses = dataService.getCollection('addresses');
    this.setData({ addresses: addresses });
  },

  saveAddress(address) {
    dataService.addToCollection('addresses', address);
  }
});
```

## 注意事项

1. **接口一致性**：当前本地存储接口与云开发接口保持一致，切换时只需修改实现
2. **数据结构**：确保本地存储数据结构与云数据库字段一致
3. **异步处理**：云开发是异步的，切换时需要处理 Promise
4. **权限设置**：云开发需要正确设置数据库权限规则

## 数据库权限规则示例

```json
{
  "read": "doc._openid == auth.openid",
  "write": "doc._openid == auth.openid"
}
```
