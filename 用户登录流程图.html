保存为JPG格式的方法：

  1. 在浏览器中按 F12 打开开发者工具
  2. 按 Ctrl + Shift + P (Windows) 或 Cmd + Shift + P (Mac)
  3. 输入 capture full size screenshot 并回车
  4. 浏览器会自动下载完整截图
  5. 或者使用截图工具（如Windows的截图工具、Mac的截图功能）手动截图

或者：

  1. 右键点击流程图
  2. 选择"检查"或"审查元素"
  3. 在开发者工具中找到包含流程图的元素
  4. 右键点击该元素，选择"Capture node screenshot"（Chrome）
